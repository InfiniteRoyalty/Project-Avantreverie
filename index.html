Personal Tips

1. MAKE MISTAKES, EXPERIMENT ART IS NOT MEANT TO BE PERFECT ALL THE TIME
2. BE CONSISTENT IF YOU’RE NOT CONSISTENT IT WILL SHOW IN THE THINGS YOU CREATE
3. USE REFERENCES THEY REALLY HELPED ME ADD EXPRESSION TO MY DRAWINGS AND IF YOU DON’T KNOW WHERE TO START WITH ART THEY ARE SUPER HELPFUL!
4. PERFECT THE THINGS YOU’RE ALREADY GOOD AT, IT WILL MAKE THINGS EASIER IF YOU KEEP DRAWING YOU WON’T HAVE AS MANY ISSUES
5. SWITCH IT UP, DON’T DRAW THE SAME THING EVERYTIME, TRY SOMETHING NEW YOU MAY END UP LIKING THAT STYLE
